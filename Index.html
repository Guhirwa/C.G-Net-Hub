Network Protocols

  • TCP/IP

    The fundamental suite of protocols for communication over the internet, including TCP (Transmission Control Protocol) and IP (Internet Protocol).

  • HTTP

    Hypertext Transfer Protocol, used for transferring web pages on the internet. The secure version, HTTPS, adds encryption.

  • FTP

    File Transfer Protocol, used for transferring files between systems over a network. Secure versions include SFTP and FTPS.

  • DNS

    Domain Name System, which translates domain names into IP addresses, allowing users to access websites using human-readable addresses.

  • SMTP

    Simple Mail Transfer Protocol, used for sending emails across networks. Works with other protocols like IMAP and POP3 for receiving and storing emails.

  • IPSec

    Internet Protocol Security, used to secure internet communications by authenticating and encrypting each IP packet.
